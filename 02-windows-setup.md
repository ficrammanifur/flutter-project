# Windows: Complete Flutter Setup from Zero

## Step 1: Install Git

Flutter requires Git to manage code repositories.

1. Visit https://git-scm.com/download/win
2. Download the installer for your Windows version (64-bit recommended)
3. Run the installer
4. Use all default options during installation
5. Restart your computer

Verify Git is installed by opening **PowerShell** (or Command Prompt) and running:

```powershell
git --version
```
